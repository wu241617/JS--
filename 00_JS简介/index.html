JS基础简介

  • JS包括，字符串，数字，布尔，null，undefined，数组，对象，symbol 等数据类型。
  • 变量作用域：作用域指在编写的算法函数中，我们能访问的变量（在使用时，函数作用域也可以是一个函数）。有本地变量和全局变量两种。
  • 在JS中有算数操作符，赋值操作符，比较操作法，逻辑操作符，位操作符，一元操作符和其他操作符。
  • 真值和假值：undefined转换成布尔值为false; null转换成布尔值为false;数字 +0,-0,NaN转换成布尔值为false,其他为true; 如果字符串长度等于0，即为空，转换成布尔值为false，其他都是true; 对象转换为布尔值为 true
  • JS的控制结构中，条件语句支持if ... else 和 switch。循环支持while,do...while,和for
